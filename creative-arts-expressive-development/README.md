# Creative Arts & Expressive Development

**Layer 1 Domain**

This domain includes visual arts, music, dance, theatre, and media arts.

It focuses on the artistic processes of Creating, Performing/Presenting/Producing, Responding, and Connecting, as outlined in the National Core Arts Standards.

**Related Strands (Layer 2 examples):**
- Creating
- Performing / Presenting / Producing
- Responding
- Connecting

See the main [TAXONOMY.md](../TAXONOMY.md) for the full 7-layer framework.
