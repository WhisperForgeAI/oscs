## About OSCS

The Open Source Curriculum Standard (OSCS) is a 7-layer taxonomy and structured framework for creating, organizing, and sharing K-12 curriculum standards in an open, collaborative, and AI-friendly way.

It is designed to be:
- Mappable to existing standards (MCCRS, CCSS, NGSS, C3, etc.)
- Flexible for competency-based / rank-based progression
- Suitable for schools, homeschooling, and AI-powered personalized learning tools

Contributions and feedback are welcome!
