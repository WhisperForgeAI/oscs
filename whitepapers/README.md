# OSCS Whitepapers

This directory contains in-depth whitepapers that explain the architectural, engineering, and research decisions behind the Open Source Curriculum Standard (OSCS).

## Purpose

These documents serve as the governing and explanatory foundation for the OSCS framework. They detail:
- Why specific design decisions were made
- Research and precedents considered
- Trade-offs evaluated
- Governance processes for future changes

| [whitepaper-taxonomy.md](./whitepaper-taxonomy.md) | OSCS 7-Layer Universal Education Taxonomy | Core taxonomical framework, layer definitions, and structural implementation |

## Future / Planned Whitepapers

- `whitepaper-architecture.md` — Technical architecture, data models, tooling, and AI integration
- `whitepaper-governance.md` — Decision-making processes, contribution guidelines, and change management
- `whitepaper-research.md` — Research foundations, comparisons to existing frameworks, and empirical rationale

## How to Contribute

Whitepapers are living documents. Proposed changes should follow the governance process outlined in future governance documents.

For now, open an issue or pull request with clear reasoning tied to the existing framework.
